# 🧪 Testes do Educampo Insights

Bem-vindo ao diretório central de testes da aplicação Educampo Insights. A nossa arquitetura de testes é dividida em diferentes escopos para garantir a qualidade, segurança e confiabilidade do sistema.

---

## 📂 Estrutura de Diretórios

- [**`security/`**](./security/README.md): Focado exclusivamente nas validações que garantem as políticas e os contratos de cibersegurança da aplicação (Proteção contra XSS, CSRF, etc).
- **`unit/`**: Contém os testes unitários da aplicação, validando o comportamento isolado das interfaces, rotas, serviços, stores e utilitários.
