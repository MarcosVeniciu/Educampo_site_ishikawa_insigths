# 🧪 Testes Unitários (`unit/`)

Este diretório contém todos os testes unitários do frontend. O objetivo é garantir que cada parte do sistema funcione perfeitamente de forma isolada, adotando uma arquitetura de testes robusta.

---

## 📂 Módulos Testados

- **`app/`**: Testes dos componentes estruturais e páginas do App Router (ex: Layouts, Dashboards).
- **`config/`**: Testes das configurações globais do sistema e roteamento (Menu/Sidebar).
- **`services/`**: Testes dos serviços de comunicação (Fachadas de Integração API/BFF).
- **`store/`**: Testes do gerenciamento de estado global da aplicação (Zustand).
- **`utils/`**: Testes utilitários, como os motores de cálculo zootécnicos e matemáticos.
